1 - Faça um programa em java que valida uma senha através de uma função "senhaCorreta" do tipo void. O main deve ter a senha (ou seja vai ter uma senha definida lá) e o usuário deverá setar ela. O usuário deve inserir a senha e caso a mesma não esteja correta deve-se lançar uma excessão (validação de senha feita através de um Try / Cacth).

2- Faça um programa que leia do usuário um valor de saque (tipo caixa eletrônico). É permitido um saque de no máximo R$ 2000,00. Caso ultrapasse esse valor deve ser lançado uma excessão. Use a função "limiteSaque".
